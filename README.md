# Orders Dashboard – Frontend

React dashboard for managing orders.

This project consumes a Django REST API and displays orders in a simple dashboard interface.

---

## Features

- Fetch orders from backend API
- Display order list
- Complete orders
- Orders statistics (total, pending, completed, cancelled)

---

## Tech Stack 

- React
- Vite
- Fetch API

---

## Project Structure
src/
├── api/
├── components/
├── pages/
└── main.jsx

---

## Backend

This frontend is designed to work with the `management_ERP_backend(django)` backend project.
